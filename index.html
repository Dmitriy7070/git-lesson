  • git clone path-to-repository
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавить новый файл под контроль гита
  • git commit - am "added new file index.html" - зафиксировать изменения и дать им комментрарий
  • git push - отдать именения на сервер (отправить все закоммиченные изменения в репозиторий на сервер)
  • git pull - забрать изменения с сервера (забрать свежие обновления)
